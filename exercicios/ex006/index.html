Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> que é uma tag não semântica

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> que não é semântica

Nesta frase, temos um termo em ênfase usando a tag <em> que é semântica

Texto marcado

Podemos também criar um texto marcado usando a tag <mark>

Com a tag <style> no cabeçalho pudemos fazer com que todos os textos marcados tivessem a mesma cor
