Repositório é um local que usamos para guardar coisas
É sempre recomendado guardar versões anteriores dos projetos que você faz
Podemos instalar em nosso computador um gerenciador de versões
O GIT é um software gratuito que faz esse controle das versões de um projeto
Uma limitação do GIT é que ele vai fazer esse controle apenas no computador em que o projeto está sendo realizado

O GitHub é uma plataforma que faz um backup das versões do projeto na nuvem de forma gratuita
O GIT é um repositório local e o GitHub é um repositório remoto
O GitHub também permite que você trabalhe em equipe
Ao ato de copiar a versão do repositório local para o remoto dá-se o nome de "Push"

Ao criar um repositório local, é recomendado que ele tenha o mesmo nome da pasta que armazena os arquivos do projeto
É recomendado que o caminho do repositório seja a pasta onde está a pasta do projeto (é o mesmo caminho da pasta do projeto)
